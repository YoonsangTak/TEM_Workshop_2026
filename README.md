### **리눅스 명령어 강의 복습**  
**경로 이동**  
**cd 디렉토리경로**: 해당 디렉토리 경로로 이동
**cd 디렉토리 첫 몇글자 입력 후 탭 누르기**: 긴 디렉토리 이름을 일일히 매번 타이핑할 필요 없이, 경우의 수가 하나로 결정되기에 필요한 글자 수만큼 입력하면 탭으로 디렉토리명 자동완성 가능.
**cd ~**: 로그인 시 접속되는 기본 경로로 이동
**cd ../**: 현재 경로의 한 단계 상위 디렉토리로 이동

**경로 내 파일 보기**  
**ls**: 현재 디렉토리에 있는 디렉토리, 파일을 이름만 보여줌.  
**ls -la**: 그냥 ls 보다 자세한 권한, 생성 날짜, 용량 등의 정보를 함께 보여줌  

**파일 출력하기**  
**cat 파일경로**: 파일을 편집기에서 여는 것이 아니라 terminal output에 출력한다.  
**grep 찾을내용 파일경로**: 파일에서 찾을 내용에 해당하는 줄을 찾아서 출력해준다. window의 ctrl+F 와 유사한 기능.  

**복사**  
**cp 복사할파일경로 붙여넣을경로** : 복사해서 붙여넣기  
**cp -r 복사할디렉토리경로 붙여넣을경로** : 해당 경로가 디렉토리일 경우 -r을 붙여서 디렉토리임을 명시해야 함.  

**삭제**  
**rm 삭제할파일경로**: 해당 경로의 파일을 삭제함  
**rm -r 경로**: 해당 경로가 디렉토리일 경우 -r을 붙여서 디렉토리임을 명시해야 함.
